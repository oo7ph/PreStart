preStart Node
=============
A basic boilerplate to get started fast on a Node REST project.


Front End
---------
- Backbone
- Bootstrap
- Createable (backbone extentions, render extentions)
- Underscore (Prototyped)
- JQuery
- Labs

Back End
--------
- Express
- Async
- Underscore
- RJS (needs to be removed)
- require-dir
